Search the web for information.

Use when:
- Finding documentation or tutorials
- Researching solutions to problems
- Getting current information beyond training data

Input:
- `query`: Search query string
- `allowed_domains`: Only include results from these domains (optional)
- `blocked_domains`: Exclude results from these domains (optional)

Returns search results with titles, URLs, and snippets.

For fetching a specific known URL, use web_fetch instead.
